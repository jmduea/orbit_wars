# Ownership (default paths)

Default touch surfaces by GitHub label `area:*`. Narrower paths may be declared in issue descriptions.

| Label | Typical paths |
| ----- | ------------- |
| `area:config` | `conf/`, `src/config/` |
| `area:training` | `src/jax/`, `src/training/` |
| `area:features` | `src/features/`, `src/jax/features.py` |
| `area:game` | `src/game/` |
| `area:artifacts` | `src/artifacts/`, `src/orchestration/` |
| `area:infra` | `scripts/`, `.github/workflows/`, `Makefile`, `.cursor/rules/` |
| `area:docs` | `docs/`, `AGENTS.md` |

Agent setup: `docs/CURSOR.md`.
