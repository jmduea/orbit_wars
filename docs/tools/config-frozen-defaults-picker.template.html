Resolved training config

Form editor for resolved Hydra training config. Pick a preset or change a config group (e.g. model=transformer_factorized_small) to load values from that YAML bundle. Each field shows which file last set it (approximate). Adjust leaves, fix warnings, then copy Hydra overrides or YAML diff.

Compositions are pre-built at generate time (no live Hydra server). Multi-group combos work when that exact bundle was embedded — otherwise use a preset or regenerate the HTML. Verify before long runs: uv run ow train print_resolved_config=true …
